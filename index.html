Aluraflix

Este projeto é uma réplica simples inspirada na interface da Netflix, desenvolvida com HTML e CSS, com o objetivo de praticar e consolidar conhecimentos em desenvolvimento front-end e design responsivo. A proposta foi criada durante meus estudos com base em cursos da Alura. Principais aprendizados: Estruturação de páginas com HTML5 Estilização avançada com CSS3 Responsividade e adaptação para diferentes dispositivos Organização de código e boas práticas em desenvolvimento web Este projeto demonstra minha capacidade de transformar ideias em interfaces funcionais e é um passo importante na minha jornada como desenvolvedor.

Meu figma

Este projeto consiste em um protótipo interativo de um catálogo de vídeos, desenvolvido no Figma, com foco em design responsivo e experiência do usuário. A interface foi projetada para dispositivos móveis, utilizando grids e componentes reutilizáveis para garantir consistência visual e funcional. Principais características: Design responsivo: Layout adaptável a diferentes tamanhos de tela. Componentes reutilizáveis: Uso de componentes para manter a consistência e facilitar futuras atualizações. Prototipagem interativa: Navegação entre telas com transições suaves, simulando a interação real do usuário. Este projeto demonstra minha habilidade em criar interfaces funcionais e esteticamente agradáveis, além de evidenciar meu domínio na utilização do Figma para prototipagem interativa.
